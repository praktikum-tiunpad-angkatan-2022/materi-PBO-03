# Access Modifier (Inheritance)

Access modifier untuk pengaksesan member class

<br>

|                   | class yang sama | package yang sama | subclass | class manapun |
| ----------------- | --------------- | ----------------- | -------- | ------------- |
| public            | ✅              | ✅                | ✅       | ✅            |
| protected         | ✅              | ✅                | ✅       | ❌            |
| default           | ✅              | ✅                | ❌       | ❌            |
| private           | ✅              | ❌                | ❌       | ❌            |
